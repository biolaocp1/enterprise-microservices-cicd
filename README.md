# Enterprise Microservices CI/CD & Observability Platform

## Project Overview
Designed and implemented a **fully automated CI/CD pipeline** for a multi-repo microservices architecture with **99.99% uptime**.

## Key Features
- GitLab CI/CD and Jenkins pipelines for multi-service deployments
- Kubernetes (RKE2) clusters with Rancher management
- Blue-green deployments and automated rollbacks
- Observability with Grafana, Prometheus, ELK, Nagios
- Infrastructure-as-Code using Terraform on AWS
- AI anomaly detection agents for predictive failure remediation

## Impact
- Deployment time reduced by 70%
- Downtime reduced by 90%
- Predictive failure detection improved system reliability

## Tech Stack
Kubernetes, Rancher, GitLab CI/CD, Jenkins, Terraform, AWS, Prometheus, Grafana, ELK, Python, AI anomaly detection

## Scripts
- `auto-remediation.py` — AI-based failure detection and auto-fix
